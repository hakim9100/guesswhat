    if ($c1Name === $c2Name) {

     return (self::ORDERS_CARDS[$c1Name]>self::ORDERS_CARDS[$c2Name]) ? +1 :-1;

Cette conditions dit tout simplement que si card1  et card2 sont egaux le resultat sera 0
